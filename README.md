# simplebrowser

simplebrowser is a very tiny web browser. It ships with an adblocker installed!

## Download

[Download is here](https://drive.google.com/drive/folders/1g8Cxx7rgD42d-fzhreqQR4afuyIzJlAT?usp=sharing)

## Building

### Windows

`npm run buildwin`

### macOS

`npm run buildmac`

### Linux

`npm run buildlinux`
